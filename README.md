# grid-splitter-react

一个极简、现代的静态 Web 工具，旨在帮助用户快速将图片裁剪为适合社交媒体展示的 3x3 (九宫格) 或 3x2 (六宫格) 布局。

## 快速开始

### 环境要求

- Node.js 20.19+ 或 22.12+
- pnpm 9.x+

### 安装与运行

1. 克隆仓库
2. 安装依赖：
   ```bash
   pnpm install
3. 启动开发服务器：
    ```bash
    pnpm run dev
    ```
4. 构建生产版本：
    ```bash
    pnpm run build
    ```
